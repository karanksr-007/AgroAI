# 🌱 AgroAI Smart Farming System

## 📌 Overview
AgroAI is an AI-powered smart farming platform that predicts crop yield based on environmental conditions and provides intelligent recommendations to improve agricultural productivity.

This system is designed to help farmers and agricultural analysts make data-driven decisions using machine learning.

---

## 🚀 Features

### 🔐 Authentication System
- User Login & Signup system
- Secure credential storage using SQLite database

### 📊 Smart Dashboard
- Real-time crop yield prediction
- Interactive UI with modern design
- Dynamic input sliders for environmental parameters

### 📈 Data Analysis
- Pie Chart (Factor Distribution)
- Bar Graph (Resource Comparison)
- Line Graph (Future Yield Trend)

### 🗺 Location Mapping
- Farm location visualization using map
- Geographic awareness for better analysis

### 🌱 Soil Intelligence
- Soil condition evaluation
- Smart suggestions based on soil quality

### 📄 AI Report Generation
- Auto-generated farming report
- Downloadable report file
- Includes prediction + recommendations

### 🤖 AI Advisory System
- Provides conclusion based on input
- Suggests improvement strategies
- Helps optimize farming decisions

---

## 🛠 Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas & NumPy
- Plotly (for interactive graphs)
- SQLite (for database)
- Joblib (model saving)
